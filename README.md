# 毎日日本語 - 매일 일본어 학습

하루 20분, 매일 일본어를 체계적으로 공부할 수 있는 웹 앱입니다.

## 🎯 기능

- **대시보드** - 오늘의 학습 진도, 연속 학습일, XP 추적
- **히라가나** - 46자 문자표, 플래시카드, 타이핑 연습
- **카타카나** - 46자 문자표, 플래시카드, 타이핑 연습
- **단어** - JLPT N5 필수 단어 (인사말, 숫자, 가족, 음식, 시간, 동사, 형용사, 일상)
- **문법** - 기초 문법 8개 패턴 (예문 포함)
- **퀴즈** - 4지선다 퀴즈 (유형/문제 수 선택)
- **20분 타이머** - 매일 학습 목표 관리

## 🚀 사용법

GitHub Pages에 배포하거나 `index.html`을 브라우저에서 열기만 하면 됩니다.

```bash
# GitHub Pages 배포
git add .
git commit -m "일본어 학습 앱"
git push
# Settings > Pages > main branch 설정
```

## 📁 파일 구조

```
├── index.html   # 메인 HTML
├── style.css    # 스타일시트
├── data.js      # 학습 데이터 (히라가나, 카타카나, 단어, 문법)
├── app.js       # 앱 로직
└── README.md
```
